# Hangingman game

Hangingman is a  guess the word game

## Installation

To build the project run the command ant inside the project folder to generate the .jar file

```bash
ant
```

## Usage

Default port: 3030

```bash
java -jar hangingman.jar
```

## Technologies used on the  project

+ Java 7
+ Java 8
+ Java Sockets
+ Java IO
